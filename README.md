# Pharmacy Mobile App

Pharmacy eshop project for client from Canada.

# Project Setup
# 1. Prequeties
- Node
- NPM
- Ionic 5
- Git

# 2. Setup
- Clone the repository
- Run 'npm install' in the Root folder

# 3. Run
- Run either of the following commands
  ionic serve
  ionic serve --labs
- Project will be opened in the browser (Preferred browser chrome)
